A Java Object-Oriented Programming project for managing hospital wards and patients. The system supports patient admission/discharge, diagnosis management, blood type filtering, occupied bed tracking, and ward administration using custom classes and arrays
# Hospital Ward Management System

This project is a Java-based Hospital Ward Management System built using Object-Oriented Programming principles.

It simulates how a hospital ward manages patients, beds, diagnoses, and departments through two main classes:

- HospitalPatient
- HospitalWard

## Features

### HospitalPatient
- Create and manage hospital patients
- Validate patient IDs and blood types
- Add and remove diagnoses
- Track admission/discharge status
- Compare patients by critical condition
- Check matching blood types

### HospitalWard
- Manage hospital wards and departments
- Admit and discharge patients
- Track occupied and available beds
- Search patients by ID
- Filter patients by blood type
- Get active patients
- Find the most critical patient in a ward
- Compare ward occupancy levels

## Concepts Used
- Classes and Objects
- Encapsulation
- Constructors
- Arrays
- Object comparison (`equals`)
- Method overriding (`toString`)

- Input validation
- Array manipulation
